# Consume

## Install for consumption

```bash
npm install @neo4j/graph-schema-utils
```

## Use

```js
import { validateSchema } from "@neo4j/graph-schema-utils";

validateSchema(theSchema);
```

# Develop / Contribute

## Install dependencies

```
npm install
```

## Run script

```
npm start
# or
node ./src/index.js
```
